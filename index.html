  • Aurras: Open-Field Active Noise Cancellation on an FPGA

    This project introduces **Aurras**, a lightweight FPGA-based system designed for open-field noise cancellation. The system dynamically reduces environmental noise by using impulse-based calibration to adapt to the acoustics of any indoor environment. It processes sound via microphones and speakers, using convolution and delay techniques to generate "anti-noise" signals that interfere destructively with unwanted sound waves. It is implemented on a Spartan-7 FPGA, optimized for low memory and computational use, and achieves significant noise reduction across a broad frequency range, especially between 500-900 Hz.
